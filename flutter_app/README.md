# flutter_app

This is a minimal Flutter project scaffold created manually.

## Setup

1. Install Flutter SDK: https://flutter.dev/docs/get-started/install
2. Open this folder in VS Code.
3. Run `flutter pub get` from the `flutter_app` directory.
4. Run `flutter run` to launch the app.

> Note: The Flutter CLI was not available in the current environment, so this project was created manually.
